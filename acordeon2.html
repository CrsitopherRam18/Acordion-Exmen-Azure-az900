EXPLORACIÓN DE SERVICIOS DE ANÁLISIS Y BASE DE DATOS

¿Qué es Azure Marketplace?

Es una tienda en línea que hospeda aplicaciones certificadas y optimizadas para ejecutarse en Azure.

¿Qué servicios ofrece Azure HDInsight?

Es un servicio de análisis de código abierto hace más fácil, rápido y rentable para procesar grandes cantidades de datos.

¿Qué es Azure Data Lake?

Es un servicio de trabajos de análisis a petición que simplifica los macrodatos ya que implementa, configura y ajusta el hardware.

¿Qué es el servicio Azure Synapse Analytics

Es un servicio de análisis ilimitado que reúne el almacenamiento de datos empresariales y el análisis de macrodatos
