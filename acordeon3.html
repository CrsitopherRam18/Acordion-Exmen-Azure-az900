EXPLORACIÓN SERVICIOS DE RED Y VISIBILIDAD DE INFORMACIÓN

¿Qué es Azure Cosmos DB?

Es un servicio de base de datos de varios modelos distribuidos globalmente, puede escalar de forma elástica e independiente el rendimiento y el almacenamiento en cualquier número de regiones Azure.

Mencione un ejemplo de servicio Iaas

Las empresas contratan la infraestructura de hardware a un tercero a cambio de una couta o alquiler, la contratación de este hardware permite elegir la capacidad de procesos.

Describe los conceptos de tipo de servicio: Iaas, Paas, Saas

Iaas: infraestructura como servicio.

Paas: Plataforma como servicio.

Saas: Software como servicio.

¿Cuál es el concepto de Azure File?

Azure Files ofrece recursos compartidos totalmente administrados en la nube a los que se puede acceder mediante los protocolos del bloque de mensajes del servidor y Network Files System
