¿Qué solución almacena objetos no estructurado permite grandes cantidades de datos de texto?

Azure Blob Storange, permite almacenar grandes cantidades de datos de texto y/o binarios.

¿Qué son las redes virtuales Azure?

Son maquinas virtuales que permiten recursos como aplicaciones web y base de datos, comunicarse entre sí, con los usuarios de internet y equipos cliente en el entorno local.

¿Cuál es la protección de redes virtuales mediante el uso de Azure firewall?

Supervisa el trafico de la red, intervalos de direcciones IP, reglas de firewall e información especifica de puertos y protocolos de red.
