CONECTIVIDAD SEGURA EN AZURE Y SERVICIOS DE IDENTIDAD

¿Qué es Active Directory?

Es un servicio que proporciona la administración de acceso e identificación administrado por su propoia configuración de acceso e identificaciones basado en la nube de Microsoft.

¿Qué es una máquina virtual windows?

Es un servicio de virtualización de escritorio y de aplicaciones que se ejecutan en la nube, permite a los usuarios usen una versión hospedada en la nube windows desde cualquier ubicación.

¿Qué ventajas tiene el servidor Azure dedicado?
  • Ofrece visibilidad y control sobre la infraestructura de servidor que ejecuta maquinas virtuales de Azure
  • Ayuda a satisfacer requisitos de cumplimiento mediante la implementación de las cargas de trabajo en un servidor aislado
  • Permite elegir el número de procesadores, capacidades, servicios de maquinas virtuales
¿Qué es Azure Database for MySQL?

Es un servicio de base de datos relacionales en la nube, y se basa em el motor de base de datos de MySQL community tiene un servicio de disponibilidad de 99,99% de Azure.
